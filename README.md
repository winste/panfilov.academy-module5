# Panfilov.academy 5 module

Hotel Booking Prototype with using Vue3, vue-router, axios, pinia, etc.

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
